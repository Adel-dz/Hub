Changement des informations personnelles:


Dans le besoin de changer vos coordonnées:

Cliquez sur le bouton Réglages, situé sur la barre des modules.


  • Dirigez-vous vers la partie intitulée "Infos. Utilisateur".
  • Effectuez les changements nécessaires.
  • Cliquez sur le bouton "Transmettre au serveur" pour finir.


Autre réglage:



Pour ceux qui sont habitués à travailler avec le code pays, cochez la case "Utiliser les codes pays". Le HUB utilisera, alors, les codes pays au lieu des noms.
