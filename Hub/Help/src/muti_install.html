Vous pouvez installer le HUB sur une autre machine mais vous ne devrez plus utiliser la copie précédente du HUB, d'ailleurs , il est recommandé  de la désinstaller. Si la copie précédente est exécutée, la copie nouvellement installée sera bloqué par le serveur central, définitivement. Si vous avez besoin d'installer le HUB sur plusieurs machines, demandez, à la sous-direction de la valeur, de vous fournir un profil pour chaque machine.

Une seule instance du HUB par profil.
